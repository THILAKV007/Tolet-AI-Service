# ==========================================
# TOLET AI
# Vision / Mission / Knowledge Base
# ==========================================

Tolet AI is an intelligent conversational
real-estate rental assistant designed to
simplify property searching using AI.

The system helps users search rental
properties naturally using:

- English
- Tamil
- Tanglish

Instead of traditional filters alone,
users can chat naturally like:

"Anna Nagar la 2BHK venum under 25k"

Tolet AI intelligently understands:

- budget
- location
- property type
- furnishing
- tenant preference
- metro preference
- conversational follow-ups

# ==========================================
# Vision
# ==========================================

To build India's smartest AI-powered
real-estate conversational platform
where users can discover rental homes
through natural conversation instead
of complicated searching.

# ==========================================
